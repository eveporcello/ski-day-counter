Ski Day Counter - Initial Setup
=================
The initial webpack build and server setup.

#### Installation
Run this npm command to install dependencies.
```
$ npm install
```

#### Build
Run this npm command to build the JavaScript Bundle
```
$ npm run build
```

#### Run
Run this npm command to build the JavaScript Bundle and open the browser to the app using the file api.
```
$ npm start
```

#### Run ESLINT
Run this command to lint the javascript source.
```
$ npm run lint
```

### Run Webpack Dev Serer
Run this command to start the app using the webpack dev server
```
$ npm run dev
```
